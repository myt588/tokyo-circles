## 公司设立

日本通常有两种公司形态：
- 株式会社
- 合同会社

这里两个公司形态区别还是蛮多的，这里不做太多的概述。

大家只需要明白一点，设立合同会社比株式会社的费用要低。而且对申请「日本经营管理签」来说两个形态都是允许的。

结下来我们就来看看，成立公司，日本国家要收取的成本费用。

|  | 株式会社 | 合同会社 |
| ---- | ---- | ----------- |
| `定款用収入印紙代` | 40,000円  <br /> （如为電子定款将不存在此费用） | 40,000円  <br />（如为電子定款将不存在此费用） |
| `定款の認証手数料` | 30,000〜50,000円 <br /> 資本金100万円未満：30,000円 <br />  資本金100万円以上300万円未満：40,000円 <br /> 資本金300万円以上：50,000円 | 0円 |
| `定款の謄本手数料` | 約2,000円 （250円/ページ） | 0円 |
| `登録免許税` | 150,000円 或者 資本金額 * 0.7% <br />（以高的费用为最终费用） | 60,000円 或者 資本金額 * 0.7% <br />（以高的费用为最终费用） |
| `合計` | 約250,000円 | 約100,000円 |

因为「日本经营管理签」对一开始不雇佣员工的公司资本金有最低要求为500万（当然如果您愿意雇佣员工，资本金可以更低）

根据资本金不同，注册公司费用也会有所不同。

那么让我们代入最低资本金，再来看一下最终成本费用

|  | 株式会社 | 合同会社 |
| ---- | ---- | ----------- |
| `定款用収入印紙代` | 0円 （基本现在都是电子定款）| 0円 （基本现在都是电子定款） |
| `定款の認証手数料` | 50,000円 | 0円 |
| `定款の謄本手数料` | 約2,000円 （250円/ページ） | 0円 |
| `登録免許税` | 150,000円 | 60,000円  |
| `合計` | 約202,000円 | 60,000円 |

[参考链接](https://www.freee.co.jp/kb/kb-launch/before-found-cost/#content1)

当然这个是交给国家的费用，注册公司环节一般会有行政书士来帮忙申请。这个酬劳也是波动比较大的。

根据谷歌的搜索结果：
- 株式会社的报酬通常在70,000〜100,000円程度、
- 合同会社在60,000〜90,000円左右。

价格参考了一篇2020年的文章，有兴趣的朋友可以谷歌翻译后阅读一下。[点我去看](https://www.biz.ne.jp/matome/2004395/#:~:text=%E4%B8%80%E6%96%B9%E3%80%81%E7%99%BB%E8%A8%98%E3%82%92%E5%90%AB%E3%82%80%E4%BC%9A%E7%A4%BE,%E3%81%AE%E3%81%8C%E4%B8%80%E8%88%AC%E7%9A%84%E3%81%A7%E3%81%99%E3%80%82)

除此之外公司成立之后还需要向税务所和社会保险事务所上交一些资料：
- 税務関係届出（※由税理士提交）一般委托费用在3万円 - 5万円之间
- 社会保険関係届出 （*由社会保険労務士提交）一般委托费用在3万円 - 5万円之间

我们把酬劳取高值，把上述价格相加我们得到：

- 株式会社：約202,000円 + 100,000円 + 50,000円 = <strong> 💴 352,000円 </strong>
- 合同会社：60,000円 + 90,000円 + 50,000円 =<strong> 💴 200,000円 </strong>

## 办公室费用

申请「日本经营管理签」是必备条件之一是一个<strong><em>独立</em></strong>的办公室。

租办公室的费用也是可高可低。先从低的讲起吧。

1. 最简单的就是租一个有独立办公空间的共享办公室。例如：

- [BizComfort](https://bizcomfort.jp/private-room-plan/) 

独立办公室费用约2万日币一个月。初期费用为：

（入会金66,000円＋清掃費11,000円）+ 20,000円 x 2（前两个月的租金）= 117,000円

2. 再来就是常规租房了。东京的最小办公室房租区间大概是：5～10万日币吧。

一般日本租办公室初期费用大概是：

2个月押金，2个月礼金，1个月中介手续费 （基本是这样，当然也可能有更便宜的）

以5万来算的话：

一开始要交出去 5万円 x 5 = 250,000円 

3. 最后就是买办公室了。这就不细说了。😊有钱的人不会在意这些细节的。

这个环节我们取个1和2的平均值：

租办公室费用：<strong> 💴 180,000円</strong>

## 签证相关费用：

签证相关基本都是给行政书士的费用。给政府的费用应该有个印紙费用4000円。除此之外基本都是人工的劳务费。

这个费用价格区间也是浮动很大。一般在15万円-40万円不等。里面的细节就不一一展开了。

我们也取个中间值：

💴 300,000円

## 总结

综上所述，如果我们把给所有必须环节的人的报酬，包括行政书士 + 税理士 + 社会保険労務士的费用都算作最高，那么综合成本会如下：

- 株式会社：352,000円 + 180,000円 + 300,000円 = <strong> 💴 832,000円 </strong>
- 合同会社：200,000円 + 180,000円 + 300,000円 = <strong> 💴 680,000円 </strong>
